== Frequently Asked Questions ==

1.What is SEO?

Answer: SEO, or search engine optimization, is the process of optimizing websites so that they rank well on search engines through organic (non-paid) searches.

2.Is this a complete SEO solution plugin?

Answer: No, this plugin does not give you all the options for extensive SEO of your posts or pages. the main purpose of this plugin is to be used in addition to those plugins which do not give the option to add 'meta name="keywords"' option to be added in the head tag.For example: Yoast SEO plugin doesn't give the option for meta keywords, so this plugin can be installed along side Yoast and you can then add keywords as well.

3.What are keywords for SEO?

Answer: Keywords are ideas and topics that define what your content is about. In terms of SEO, they're the words and phrases that searchers enter into search engines, also called "search queries", and get relevant results.

4. Are Metadata Keywords required for SEO?

Answer: Adding Metadata Keywords is good and standard practice. Although some may argue that it is irrelevant, it definitely isn't something that is totally ignored by search engines. A good Search Engine Optimization service will always include the metadata keywords in the head tag.
